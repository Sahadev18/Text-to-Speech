# Text-to-Speech
A text to speech web application to convert the user entered text into speech

step 1: Click on code in the repository to get the link of the project.
step 2: In terminal, in any directory initialize a repository using 'git init' command. (for this you need to have git installed on your local computer)
step 3: After initializing, use the 'git clone <url>' command to clone the complete project on your local computer. (this will download the complete project in the directory that you initialized git in.)
step 4: Tools required for running the project:
          a: Nodejs
          b: Visual studio code
          c: Xampp server
step 5: Open the project in Visual studio code.
step 6: Open the terminal in Vs code and download the following dependencies using the command 'npm i <dependency>':
          a: bcrypt
          b: cookie-parser
          c: dotenv
          d: express
          e: hbs
          f: jsonwebtoken
          g: mysql
          h: nodemon
step 7: Open xampp server and create a database with any name, in the same directory create a table with any name and the attributes in the table should be
          a: name
          b: email
          c: phno
          d: password
          (note that the above attributes should be written as it is, if changed should be changed in the code also.)
step 8: Create a new file in the same directory named as '.env'.
